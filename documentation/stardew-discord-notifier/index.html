Stardew Discord Notifier

Never miss a morning! Get daily weather, luck, birthdays, and festival updates from Stardew Valley directly to your Discord channel.

Nexus Mods CurseForge Setup Guide
v1.0.2 SMAPI 4.0+ Stardew 1.6+

Main Features

Weather Forecast

Know instantly if it's going to be sunny, rainy, or stormy to plan your farming or watering duties ahead of time.

Daily Luck

Get the TV fortune teller's report delivered to you. Perfect for deciding if it's a good day for a mining run.

Birthdays & Festivals

Never miss a villager's birthday or an important seasonal festival again with automatic reminders.

Financial Status

Track your current funds with every morning report. This feature can be toggled on or off.

Multiplayer Ready

Optionally list all online players in the notification. Great for keeping track of who is on the farm.

Embed & Text Modes

Choose between a stylish Discord Embed box or a simple plain text message to suit your server's style.

Installation

1

Install SMAPI

Ensure you have the latest version of SMAPI installed.

2

Download the Mod

Download and unzip the mod files. Place the StardewDiscordNotifier folder into your game's Mods directory.

3

Configure (Important!)

Run the game once to generate the config file, or use Generic Mod Config Menu in-game to set your Webhook URL.

Configuration

How to get a Webhook URL

  1. Open your Discord Server.
  2. Right-click the channel you want notifications in and select Edit Channel.
  3. Go to Integrations > Webhooks.
  4. Click New Webhook. Customize the name and avatar if you wish.
  5. Click the Copy Webhook URL button.
  6. Paste this URL into the Webhook URL field in the mod settings.

Configuration Options

Webhook URL Required
Bot Name / Avatar Customize the sender identity
Use Embed Mode Toggle fancy box display
Show Players List online players (Multiplayer)
Toggle Info Show/Hide Date, Weather, Money, etc.

We highly recommend installing Generic Mod Config Menu for easy in-game configuration.
